# 🛒 E-Commerce Sales Analysis

A complete data analysis project using **Python, Pandas, Matplotlib, Seaborn, and SQL** to analyze 1000+ retail sales transactions and extract actionable business insights.

---

## 📌 Project Overview

This project simulates a real-world e-commerce sales dataset and performs end-to-end analysis including data cleaning, visualization, and SQL-based querying to help business teams make data-driven decisions.

---

## 🛠️ Tools & Technologies

| Tool | Purpose |
|------|---------|
| Python | Core programming language |
| Pandas | Data loading, cleaning, transformation |
| Matplotlib | Chart creation |
| Seaborn | Advanced visualizations |
| SQLite | SQL queries inside Python |

---

## 📊 Charts & Visualizations

| Chart | Description |
|-------|-------------|
| `chart1_category_revenue.png` | Total revenue by product category |
| `chart2_monthly_trend.png` | Monthly sales revenue trend |
| `chart3_top_customers.png` | Top 10 customers by total spend |
| `chart4_city_avg.png` | Average order value by city |

---

## 🗄️ SQL Queries Covered

1. Revenue by product category
2. Monthly revenue trend
3. Top 5 customers by spend
4. City-wise performance (revenue + avg order value)
5. Top category-city combinations (cross analysis)

---

## 💡 Key Business Insights

- **Electronics** contributed the highest total revenue among all categories
- **Repeat customers** (ordering more than once) accounted for the majority of total orders
- City-wise analysis revealed which markets have the highest average order value
- Cross-analysis of category × city helps decide where to focus inventory and marketing

---

## 🚀 How to Run

**Step 1 — Install dependencies**
```bash
pip install pandas matplotlib seaborn
```

**Step 2 — Run the script**
```bash
python ecommerce_analysis.py
```

**Step 3 — Output**
- 4 chart PNG files will be saved in your current folder
- SQL query results will print in the terminal
- Key insights summary will display at the end

> No CSV file needed — the dataset is generated inside the script itself.
